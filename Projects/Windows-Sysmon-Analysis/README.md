# Windows Event Logging & Sysmon Analysis

## Overview

This project demonstrates deployment of Sysmon and analysis of Windows process creation and process termination events using Event Viewer.

## Objectives

- Install and configure Sysmon
- Generate Windows process activity
- Investigate Event ID 1 (Process Creation)
- Investigate Event ID 5 (Process Termination)
- Understand parent-child process relationships

## Tools Used

- Windows 11
- Sysmon
- Event Viewer
- PowerShell
- Windows Terminal

## Key Findings

- Sysmon successfully captured process creation events.
- Process termination activity was recorded and analyzed.
- Parent-child process relationships were identified.
- PowerShell and Notepad activity was monitored through Sysmon logs.
