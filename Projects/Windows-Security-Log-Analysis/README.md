# Windows Security Log Analysis

## Overview

This project focuses on investigating authentication, privilege assignment, and credential-related activities recorded within Windows Security Logs.

## Objectives

- Analyze Windows Security Logs
- Investigate authentication events
- Review failed logon attempts
- Examine privileged account activity
- Monitor Credential Manager access

## Event IDs Investigated

| Event ID | Description                 |
| -------- | --------------------------- |
| 4624     | Successful Logon            |
| 4625     | Failed Logon                |
| 4672     | Special Privileges Assigned |
| 5379     | Credential Manager Access   |


## Tools Used

- Windows 11
- Event Viewer

## Key Findings

- Successful and failed authentication events were identified.
- Privileged account activity was reviewed.
- Credential Manager interactions were observed.
- Security logs provided visibility into account usage patterns.

- Security Monitoring
